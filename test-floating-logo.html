Test Page for Noma Floating Logo

The Noma logo should appear in the top-left corner of this page.

Open the browser console (F12) to see debug messages.

Instructions:

  1. Make sure the Noma extension is loaded in Chrome
  2. Go to chrome://extensions/
  3. Find "Noma" and click the reload button (circular arrow icon)
  4. Come back to this page and refresh (Cmd+R or Ctrl+R)
  5. Check the top-left corner for the logo
  6. Open DevTools Console (F12) to see debug logs
